## 使用方法

1. 你需要在你的设备上安装`Python3`，并通过`pip`安装好`pyftpdlib`
2. 打开本App，点击右下角设置，输入`Python`的路径
3. 选择路径，点击启动服务下面的开关即可使用

注意，默认的FTP服务端口号为`2121`
